# Prefixes & notations
Prefixes and notations I use for various apps and tools and their meanings.

## SnippetsLab
- <> - User input

## Pixave
- w. - Wallpapers
- r. - Reactions
- s. - Scans
- rg. - Reaction GIFs

## Contacts
- () - Nickname
- \# - Tags

## PDFs
- r. - Research papers
- d. - Documents
- b. - Books
- s. - Personal

## Keyboard Maestro
- {} - What should be in my clipboard
- [] - What state the macro should be in
- <> - What link should be in my browser

## Safari
- w: - Websites
- s: - Personal
- g: - GitHub
- c: - Chat
- a: - Analytics
